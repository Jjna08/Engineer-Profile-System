# Engineer Profile System

## Description
A recruitment website that displays engineer profiles, allows searching by name, viewing details, and hiring engineers.

## Technologies
- HTML
- CSS
- JavaScript
- LocalStorage

## Features
- Search engineers by name
- Display 8 random engineers on every refresh
- Engineers sorted alphabetically
- Popup with engineer details
- Hire / Unhire engineer
- Data saved using LocalStorage

## How to Run
1. Download the project.
2. Open the project folder in VS Code.
3. Install the Live Server extension.
4. Right-click `index.html`.
5. Click **Open with Live Server**.
